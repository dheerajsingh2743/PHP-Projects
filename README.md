# 🧾 PHP CRUD Operation Project

## 📌 Project Description

This is a simple PHP CRUD (Create, Read, Update, Delete) application.
It allows users to manage data such as adding new records, viewing records, updating existing data, and deleting records.

---

## 🚀 Features

* ➕ Add new data (Create)
* 📋 View all records (Read)
* ✏️ Update existing data (Update)
* ❌ Delete records (Delete)
* 🔍 Simple and user-friendly interface

---

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML
